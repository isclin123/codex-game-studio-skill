# Game Jam Workflow

Use for 24-hour, 48-hour, 72-hour, and one-week jam projects.

## Timebox

- First 10 percent: pick idea and platform.
- Next 40 percent: make the core loop playable.
- Next 30 percent: add feedback, juice, and one content arc.
- Final 20 percent: bugs, restart, title, instructions, build, submission.

## Scope Rules

- One verb, one twist, one ending.
- Avoid online multiplayer unless the jam theme demands it.
- Use simple assets and strong silhouettes.
- Prefer web builds for easy judging.

## Submission Checklist

- Controls visible.
- Restart works.
- Audio can be muted.
- Build tested on a clean machine/browser.
- Page includes screenshots, short description, and known issues.
