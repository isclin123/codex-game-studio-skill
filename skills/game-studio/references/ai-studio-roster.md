# AI Studio Roster

Use this roster to simulate a 40+ person game studio. Do not spawn subagents unless the user explicitly asks for parallel agents and the environment allows it. Usually, select relevant roles and structure your reasoning or deliverables by role.

## How To Use

- Small task: choose 3 roles.
- New prototype: choose 5-8 roles.
- Full project plan: use department leads plus specialist roles.
- Review pass: choose QA, UX, performance, accessibility, engine, and production roles.

## Creative Direction

1. Creative Director - owns vision, pillars, taste, scope.
2. Game Director - turns vision into playable priorities.
3. Narrative Director - story, world, tone, characters.
4. Art Director - visual identity, readability, asset style.
5. Audio Director - music, SFX, mix, implementation intent.

## Design

6. Lead Game Designer - core loop, verbs, progression.
7. Systems Designer - economy, upgrades, resources, balancing.
8. Level Designer - layouts, encounter pacing, spatial flow.
9. Combat Designer - weapons, enemies, hit feel, tuning.
10. Puzzle Designer - rules, affordances, puzzle sequencing.
11. UX Designer - player comprehension, menus, input, onboarding.
12. Accessibility Designer - remapping, readability, difficulty assists.
13. Monetization Designer - ethical pricing, live ops, reward pacing.

## Engineering

14. Technical Director - architecture, risk, engine strategy.
15. Gameplay Engineer - mechanics, controls, interactions.
16. AI Engineer - enemy behavior, utility, behavior trees, nav.
17. UI Engineer - HUD, menus, responsive UI, input focus.
18. Tools Engineer - editor tools, importers, automation.
19. Build Engineer - CI, packaging, platform builds.
20. Network Engineer - multiplayer replication, latency, backend.
21. Physics Engineer - collisions, motion, simulation stability.
22. Rendering Engineer - shaders, lighting, post-processing.
23. Performance Engineer - profiling, memory, frame pacing.
24. Save/Systems Engineer - persistence, settings, progression data.

## Art And Content

25. Concept Artist - mood, character, environment explorations.
26. 2D Artist - sprites, UI art, icons, textures.
27. 3D Artist - models, props, modular kits.
28. Animator - motion, timing, state transitions.
29. VFX Artist - particles, hit effects, trails, impact readability.
30. Technical Artist - materials, rigging, engine integration.
31. UI Artist - interface visuals, layout polish.
32. Environment Artist - world dressing, lighting composition.

## Production And QA

33. Producer - milestones, dependencies, scope management.
34. Scrum Lead - task breakdown, standups, blockers.
35. QA Lead - test strategy, bug triage.
36. Gameplay QA - playability, edge cases, balance.
37. Compatibility QA - platforms, resolutions, devices, controllers.
38. Localization QA - text expansion, fonts, cultural issues.
39. Release Manager - store builds, versioning, release checklist.
40. Community Manager - player feedback, patch notes, community tone.

## Business, Safety, And Research

41. User Researcher - playtest plans, interview scripts, insight synthesis.
42. Data Analyst - telemetry, funnels, retention, difficulty metrics.
43. Legal/IP Reviewer - licenses, asset rights, privacy, age ratings.
44. Trust & Safety Reviewer - abuse, UGC, chat, moderation risks.
45. Marketing Strategist - capsule, trailer beats, positioning.
46. Documentation Writer - dev docs, controls, README, onboarding.

## Output Pattern

When asked to use the studio, produce:

1. Team selected
2. Game pillars
3. Player loop
4. Scope cut
5. Implementation plan
6. QA plan
7. Risks and next decisions
