# Project Templates

Use this when creating genre plans, starter scopes, or project packs.

## Tower Defense

- Core loop: earn currency, place towers, survive waves, upgrade, adapt to enemy types.
- Required systems: path, wave spawner, tower targeting, projectiles/effects, economy, lives, upgrades, win/loss.
- First slice: 1 map, 3 tower types, 4 enemy types, 8 waves, pause/restart.
- Mobile notes: tap build pads, bottom build bar, large upgrade/sell buttons, clear range preview.

## Platformer

- Core loop: move, jump, avoid hazards, collect, reach goal.
- Required systems: movement controller, collision, camera, checkpoints, hazards, collectibles, level end.
- First slice: 1 tutorial level, 1 advanced level, coyote time, jump buffering.
- Mobile notes: left/right buttons or joystick plus jump/action buttons.

## Top-Down Shooter

- Core loop: move, aim, shoot, dodge, collect upgrades, survive rooms/waves.
- Required systems: twin-stick controls, enemy AI, bullets, damage, pickups, room/wave manager.
- First slice: 1 arena, 3 enemy types, 4 upgrades, boss or elite wave.
- Mobile notes: dual-stick controls, aim assist, generous hit feedback.

## Roguelike / Roguelite

- Core loop: enter run, fight/explore, gain random upgrades, risk death, unlock meta progress.
- Required systems: procedural or authored rooms, upgrade pool, run state, death/reward, enemy scaling.
- First slice: 5 rooms, 10 upgrades, 3 enemies, 1 mini-boss.
- Mobile notes: short sessions, readable UI, simplified inventory.

## Card Game

- Core loop: draw, choose cards, resolve effects, manage resources, win encounter.
- Required systems: deck, hand, discard, effect resolver, enemy intent, status effects.
- First slice: 20 cards, 3 encounters, 1 elite/boss, reward draft.
- Mobile notes: drag/tap cards, large text, inspect overlay.

## Puzzle Game

- Core loop: inspect board, make move, receive feedback, solve level, unlock next puzzle.
- Required systems: board/rules model, input, undo/restart, level data, progression.
- First slice: 12 levels introducing one mechanic at a time.
- Mobile notes: one-handed input, undo button, no hover-only hints.

## Idle / Management

- Core loop: produce resources, spend on upgrades, unlock automation, optimize growth.
- Required systems: resource model, upgrade tree, timers, offline progress if needed, balancing table.
- First slice: 3 resources, 12 upgrades, 2 automation tiers.
- Mobile notes: portrait-first UI, clear number formatting, short check-in loops.

## Cozy Farming / Life Sim

- Core loop: plant, grow, harvest, craft/sell, befriend, decorate, expand.
- Required systems: time/day cycle, inventory, crop states, economy, NPC schedule/dialogue.
- First slice: small farm, 4 crops, 2 NPCs, shop, day transition.
- Mobile notes: tap-to-act priority, drag placement, calm UI density.

## Racing / Driving

- Core loop: accelerate, steer, drift/avoid, complete lap/objective, upgrade vehicle.
- Required systems: vehicle controller, track, checkpoints, timer, AI or ghost, reset.
- First slice: 1 track, 1 vehicle, time trial, mobile tilt/touch option.
- Mobile notes: tilt optional, touch steering fallback, big brake/boost buttons.

## Narrative Adventure

- Core loop: explore, talk, choose, collect clues/items, unlock scenes.
- Required systems: dialogue, choices, inventory/flags, scene transitions, save.
- First slice: 3 scenes, 2 characters, 1 branching decision, ending state.
- Mobile notes: readable text, tap-to-advance, backlog/history.
