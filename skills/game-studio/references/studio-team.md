# Studio Team Workflow

Use for multi-person teams, production planning, reviews, and role coordination.

## Team Lanes

- Design: pillars, mechanics, levels, balance.
- Engineering: architecture, gameplay systems, tools, platform.
- Art: style, assets, animation, technical constraints.
- Audio: SFX, music, implementation, mix.
- QA: test plans, bug triage, regression.
- Production: milestones, risks, dependencies, scope.

## Planning

- Keep a shared definition of done.
- Break work by vertical slices, not isolated departments.
- Track dependencies between assets, code, scenes, audio, and QA.
- Review risks weekly.

## Handoff Standards

- Every task needs owner, acceptance criteria, test step, and rollback note.
- Art/audio assets need naming, import settings, and implementation target.
- Engineering changes need run steps and test status.
